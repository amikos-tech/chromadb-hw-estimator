# ChromaDB Hardware Estimator

Estimate the hardware requirements based on your data.

## Installation

```bash
pip install chromadb-hardware-estimator
```

## Usage

```bash
chromadb-hwest --help
```

```bash
chromadb-hwest calculate --data-path /path/to/data --data-type <pdf|text> --model <openai|...>
```
